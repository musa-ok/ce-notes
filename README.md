# 📚 CS-Fundamentals

Bu depo, bilgisayar mühendisliği temelleri, yazılım geliştirme araçları ve temel programlama konseptleri üzerine aldığım notları ve yaptığım küçük çaplı uygulamaları içermektedir.

## 📂 İçerik Yapısı

| Klasör / Dosya | Açıklama |
| :--- | :--- |
| **`oop/`** | Nesne Yönelimli Programlama (Decorator, Generator, Sınıf yapıları vb.) |
| **`Sqlite/`** | Veritabanı yönetimi ve Python ile SQLite entegrasyonu çalışmaları. |
| **`git/`** | Versiyon kontrol sistemleri (Git) üzerine temel komutlar ve iş akışı notları. |
| **`Scripts/`** | `Encode.py`, `Qdrant.py` gibi yardımcı Python scriptleri. |

## 🛠️ Kullanılan Teknolojiler & Araçlar
* **Dil:** Python 3.x
* **Veritabanı:** SQLite
* **Araçlar:** Git, VS Code
